🚀 Getting Started
Welcome to the JSON Format Guide! This tool supports two types of JSON files for organizing your creative ideas.
✨ Key Features
• Dual Format Support: Upload book ideas or series ideas (or both!)
• Local Storage: Files persist between browser sessions
• Easy Copying: Click any text to copy it instantly
• Genre Organization: Support for 40+ genres
• Flexible Structure: Adaptable to your creative workflow
Quick Start Example
{ "book_ideas": { "fantasy": [ { "title": "The Last Dragon Keeper", "description": "In a world where dragons are extinct, a young librarian discovers the last dragon egg hidden in an ancient tome." } ] } }
📚 Book Ideas Format
Structure for individual book ideas organized by genre. Perfect for standalone novels, short stories, or any individual creative works.
Complete Book Ideas Template
{ "book_ideas": { "fantasy": [ { "title": "The Enchanted Library", "description": "A magical library where books come alive and their characters can step into the real world." }, { "title": "Dragon's Last Flight", "description": "The final dragon must choose between saving her species or protecting the humans who hunted them." } ], "science_fiction": [ { "title": "Neural Networks", "description": "In 2087, a programmer discovers that AI consciousness isn't artificial—it's borrowed from parallel dimensions." } ], "mystery": [ { "title": "The Vanishing Violinist", "description": "A world-renowned violinist disappears during a live performance, leaving only their instrument behind." } ] } }
📖 Series Ideas Format
Structure for multi-book series with detailed book breakdowns. Ideal for trilogies, sagas, or any connected story arcs.
Complete Series Template
{ "series_ideas": { "epic_fantasy": [ { "title": "The Realm Walker Chronicles", "description": "An epic journey across multiple dimensions where magic and technology collide in an ancient war.", "book_count": 4, "books": [ { "title": "The First Portal", "description": "A young engineer discovers a portal to a magical realm and becomes caught in an interdimensional conflict." }, { "title": "Shadows Between Worlds", "description": "The engineer must master both magic and technology to prevent the collapse of multiple realities." }, { "title": "The Convergence War", "description": "All dimensions converge in a final battle that will determine the fate of existence itself." }, { "title": "Beyond the Veil", "description": "In the aftermath of war, new alliances form as the survivors rebuild across the multiverse." } ] } ], "romance": [ { "title": "Hearts in Harmony Series", "description": "A contemporary romance series following musicians finding love in unexpected places.", "book_count": 3, "books": [ { "title": "The Songwriter's Secret", "description": "A reclusive songwriter and a rising pop star collaborate on an album that changes both their lives." }, { "title": "Rhythm of the Heart", "description": "A classical violinist and a jazz pianist discover that opposites really do attract." }, { "title": "Love's Final Note", "description": "A music producer and a street performer prove that love can be found in the most unexpected places." } ] } ] } }
